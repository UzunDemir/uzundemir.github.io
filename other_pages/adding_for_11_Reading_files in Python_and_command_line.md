Динамичное развитие. С каждым годом производительность языка растёт, как и его огромное сообщество, которое расширяет базовые возможности сторонними библиотеками.

# Вопрос 2
Какие встроенные типы данных доступны в Python?

В Python есть изменяемые и неизменяемые встроенные типы.

кортежи,
числа.

# Вопрос 3
Что такое изменяемые и неизменяемые типы данных?

## *Ответ 3*
Все объекты в Python относятся к одной из двух групп: неизменяемым и изменяемым.

Неизменяемые типы данных в конце программы выглядят так же, как в начале. К ним относятся числа, строки и кортежи. 
Изменяемые типы данных могут модифицироваться в процессе работы программы. Например, в них могут добавляться и удаляться элементы. К данным такого типа относятся списки, множества и словари. 

Когда данные передаются в функцию, способ их обработки зависит от типа. Например, для неизменяемых чисел создаётся независимая копия, следовательно, любое преобразование внутри функции не повлияет на исходное число. И наоборот, вместо изменяемого списка передаётся указатель на то место в памяти, где он хранится, и все трансформации влияют на внешний объект.

# Вопрос 4
Чем отличаются списки и кортежи?

## *Ответ 4*
В Python кортежи и списки — это встроенные структуры данных. Вот несколько различий между списком (list) и кортежем (tuple):

Синтаксис. Кортеж заключается в круглые скобки, а список — в квадратные.
Изменяемость. Кортеж — это неизменяемая структура данных, а список — изменяемая.
Размер. Кортеж в Python занимает гораздо меньше места, чем список.
Производительность. Кортеж быстрее, чем список. Это даёт хорошую производительность.
Перед выбором используемой структуры вы должны задать вопрос: должна ли она изменяться? Если да, используйте список, если нет, используйте кортеж.

Для коллекций разнородных объектов (например, адрес, разбитый на составляющие: Ф. И. О., улица, город, индекс) лучше подойдёт кортеж. Используйте кортеж, если вам нужен контейнер, содержащий множество объектов.

Также кортежи, как правило, используются там, где порядок и положение объекта значимы и постоянны. Например, при создании структуры для игры «Подземелья и драконы» можно использовать кортежи вместо списков, так как позиция объекта в кортеже имеет значение. Небольшой пример:

pages = {
    'Коридор': {
        'Текст': 'Какой-то текст...',
        'Выбор': [
            ('Открыть дверь', 'Радуга!'),
            ('Вернуться в кухню', 'Бездонная яма...'),
            ('Остаться на месте', '...'),
        ],
    },
}
Первая позиция в кортеже — выбор, отображаемый пользователю во время игры, а вторая — ключ страницы, на которую переходит игрок, и такое положение одинаково для всех страниц.

# Вопрос 5
Что может быть ключом в словаре? А что будет значением?

## *Ответ 5*
В Python ключом словаря может быть любой неизменяемый тип данных: целые и действительные числа, строки, кортежи. Ключом в словаре не может быть множество, но может элемент типа frozenset — специальный тип данных, аналог типа set, но без возможности изменять его после создания. 

Значением элемента словаря может быть любой тип данных, в том числе изменяемый.

# Вопрос 6
Что такое PEP8 и для чего он нужен? Насколько важна читабельность кода?

## *Ответ 6*
PEP8 — это документ, содержащий рекомендации по написанию кода на Python. Основная цель PEP8 — улучшить читабельность и логичность кода.

Гвидо ван Россум сказал: «Код читают гораздо чаще, чем пишут». Вы можете потратить несколько минут или целый день на написание фрагмента кода для загрузки файла. В дальнейшем вам не придётся его писать, но вы точно будете его перечитывать. Этот фрагмент кода может остаться частью рабочего проекта, и каждый раз, когда вы будете возвращаться к этому файлу, нужно будет вспомнить, что делает этот код и зачем вы его написали. Поэтому удобочитаемость очень важна.

# Вопрос 7
Что означает пространство имён в Python?

## *Ответ 7*
Пространство имён — это система имён, которая обеспечивает уникальные наименования всех объектов в программе, чтобы избежать возможных конфликтов. 

В Python пространства имён реализованы как словари с именем в качестве ключа и объектом в качестве значения. В результате разные пространства могут давать своим объектам одинаковые имена.

В Python три типа пространств имён:

Локальное пространство имён — включает локальные имена внутри функции, временно создаётся во время вызова функции и очищается при возврате из неё.
Глобальное пространство имён — состоит из имён импортированных пакетов или модулей, которые используются в проекте. Создаётся при импорте пакета в скрипт и доступно до тех пор, пока скрипт не закончит выполнение.
Встроенное пространство имён — включает встроенные функции Python и встроенные имена для различных типов исключений.

# Вопрос 8
Как бороться с ошибками в Python? Как работает механизм исключений?

## *Ответ 8*
При анализе или исполнении кода иногда возникают ошибки или исключения. Это может произойти при попытке чтения файла, которого не существует. Хороший разработчик должен уметь ловить и обрабатывать такие ошибки.

В Python определён главный класс BaseException, от которого наследуются все остальные классы ошибок. У него есть четыре прямых наследника, в том числе Exception — класс, откуда наследуется больше десятка различных ошибок, которые может обработать программист.

Чтобы поймать и обработать исключения, нужно использовать конструкцию try — except — finally. В блоке try размещается код, который должен быть выполнен. Блок except даёт возможность поймать и обработать нужные ошибки. При необходимости можно добавить инструкцию finally, которая выполнится в любом случае.

# Вопрос 9
В чём разница между оператором is и оператором сравнения ==?

## *Ответ 9*
Оператор is проверяет идентичность объектов (одинаковые адреса в памяти), а == — равенство значений, которые принимают объекты.

# Вопрос 10
Зачем использовать оператор pass в Python?

## *Ответ 10*
Оператор pass используется как заполнитель для будущего кода. При выполнении ничего не происходит, но вы избегаете ошибки, если пустой код не разрешён. 

Пустой код не допускается в циклах, определениях функций и классов, операторах if.

# Вопрос 11
Как работает алгоритм бинарного поиска?

## *Ответ 11*
В большинстве случаев поиск применяется к упорядоченным данным. Эффективный алгоритм для таких случаев — поиск делением пополам. В этом методе сравнение осуществляется с элементом, расположенным в середине диапазона. В зависимости от результата сравнения (больше или меньше) дальнейший поиск проводится в левой или правой половине диапазона аналогичным образом.

# Вопрос 12
Опишите любой алгоритм сортировки.

## *Ответ 12*
Для примера возьмём алгоритм пузырьковой сортировки. 

В процессе его работы происходит попарное сравнение соседних элементов списка. Если первый элемент больше второго, то они меняются местами. Чтобы избежать сбоя или ошибки, осуществляется несколько таких проходов со сравнением элементов в массиве. Бо́льшие элементы постепенно передвигаются в конец списка, а меньшие — в начало. Получается «пузырьковый эффект», где более «лёгкие» элементы устремляются на поверхность, а «тяжёлые» опускаются на дно.

# Вопрос 13
Что такое лямбда-выражение в Python?

## *Ответ 13*
Лямбда-выражение в Python используется для создания анонимной функции, которая может принимать любое количество аргументов. Однако лямбда-функция может иметь только одно выражение или инструкцию. Обычно она используется, когда требуется анонимная функция в течение короткого периода времени.

Пример лямбда-функции, которая складывает два числа:

custom_func = lambda x, y: 2*x + 3*y
result = custom_func(10, 20)
print(result) # Вывод: 80

# Вопрос 14
Что такое модуль в Python?

## *Ответ 14*
Модуль — это файл, написанный на Python и содержащий код с инструкциями импорта, классами и функциями. 

Модули можно импортировать в другой файл Python. С помощью модулей функциональность приложений может быть разделена для структурирования программы.

Папка программы Python — это пакет модулей. Такой пакет может содержать модули или подпапки.

# Вопрос 15
Что такое None в Python?

## *Ответ 15*
Ключевое слово None в Python представляет нулевые значения. Объект None имеет тип NoneType. С использованием таких объектов может быть выполнена операция логического равенства.
